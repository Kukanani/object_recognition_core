object_recognition_core for developers
######################################

Adding a pipeline to ``ORK`` is done by wrapping code and is usually around
200/300 lines of Python/C++. You should read the reference documentation (see
the :ref:`docs homepage <index>`) first, so that you know what is expected
of your pipeline.
